## ðŸŽ® Gameplay

### Core Loop
1. Choose your **Flavour Fighter**  
2. Fight in a physics-based arena (60 seconds)  
3. Win by:
   - Depleting enemy HP  
   - Knocking them out of the arena  
   - Or having more HP when time runs out  
4. **The Fighter will only hit the side which you are facing!!**
